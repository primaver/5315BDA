What is Docker

Docker is tool that is able to create, deploy, and run application in small compacted files known as containers. These containers are called upon when needing to be used, which allows for seamless programming without overusing computational space / data.

https://opensource.com/resources/what-docker

What is Nginx

Nginx is an open source software that provides a platform for software to be ran, communications with multiple servers, store programming for a later use, distributes incoming workload across multiple computing resources, etc.

What is VM (virtual machine)

A virtual machine (VM) is a file that acts and functions like a normal computer. The VM is able to run applications, and provides a similar user interface. Also, the virtual machine is lock safe, which means that it cannot be contaminated nor infect other machines.

https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/
