What is Git?

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. This acts as a medium between the local and remote repository, with the capability of pulling previous versions and comparing program through simple commands.

https://git-scm.com/

Why do we need source code Version Control System?

Its important to have source code version control system in order to record changes to a file or set of files over time. This allows for a capability to recall specific versions for any reasons bugs and/or erros occur within the current version of code. Additionally, the version control allows for analyzing changes over time and/or what introduce a new error, etc.

https://git-scm.com/book/en/v2/Getting-Started-About-Version-Control
